InstagramAPI

Instagram API - Feed

Het onderstaande voorbeeld maakt gebruik van HTML, CSS, JavaScript, jQuery, Ajax en de InstagramAPI. Om toegang te krijgen tot de InstagramAPI moet je gebruik maken van de developer omgeving van Facebook (Meta).

Het clonen of doornemen van deze repository geeft je al toegang tot veel code die je op de juiste weg helpt, de rest mag je mooi zelf uitzoeken.
